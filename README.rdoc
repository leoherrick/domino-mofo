= Domino Mofo
Ruby command-line dominos game

This is a fun command-board dominos game I developed in the BDD manner using Cucumber and Rspec.

== To Play

Download and run:
  bin/domino_mofo

=== Commands

* Lead <domino>
* Play <domino> on <domino> by <suit>
* score
* count
* board
* hand
* knock
* draw
* exit

== To Run Rspec

Download and run:
  rspec spec
